CONFIGURAR BACK

2. importar el proyecto que se encuentra en la carpeta BACK
3. entrar al archivo. la ruta C:\Users\Sergio\Desktop\ejercicio\project\src\main\webapp\WEB-INF\web.xml

1. abrir el IDE(visual studio code)
2. importar el proyecto que se encuentra en la carpeta FRONT
3. en la consola del visual studio code ejecutar npm start
4. para evitar errores de Allow-Control-Allow-Origin instalar el complemento de chrome con el mismo nombre y activarlo
